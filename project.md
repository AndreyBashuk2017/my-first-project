# Мой первый проект

Персональный веб-проект Андрея Башука. Включает личную визитку и лендинг для строительной компании. Все страницы написаны на чистом HTML/CSS/JS без фреймворков и внешних зависимостей.

---

## Для кого

- Личная визитка — для знакомства с Андреем как специалистом
- Лендинг — для привлечения клиентов на строительство загородных домов из газоблока (сегмент среднего класса)

---

## Страницы проекта

| Файл | Назначение |
|------|------------|
| `vizitka.html` | Личная визитка — контакты, краткое описание, ссылки |
| `landing.html` | Лендинг строительной компании — преимущества, аудитория, форма заявки |

## Стиль дизайна

- **Тёмная тема** — фон `#0a0a0a` / `#0d1117`
- **Градиенты** — акцентные элементы и заголовки оформлены градиентами (синий → фиолетовый → бирюзовый)
- **Glassmorphism-карточки** — полупрозрачные карточки с тонкой рамкой
- **Анимации** — плавное появление элементов, пульсация CTA-кнопки, hover-эффекты
- **SVG-иконки** — встроены прямо в HTML, без внешних библиотек
- **Адаптивность** — все страницы корректно отображаются на телефоне
